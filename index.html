Frequently asked questions

Where my scripts located and how it’s secured?

Your JavaScript code is securely stored in an encrypted format in your browser.

What if I want to copy tools to another PC?

Every registered user has their unique code that can be shared to copy their tools.

What if I want to run code on another language?

Version 1.0.0 does not have the ability to change the language, but it’s may be possible in the next time.

Is this extension available for all websites?

The extension runs scripts on websites with the corresponding URL and does not support local sources.
